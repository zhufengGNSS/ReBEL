gppak

PURPOSE

GPPAK Combines GP hyperparameters into one vector.

SYNOPSIS

function hp = gppak(net)

DESCRIPTION 

GPPAK    Combines GP hyperparameters into one vector.

    Description
    HP = GPPAK(NET) takes a Gaussian Process data structure NET and
    combines the hyperparameters into a single row vector HP.
